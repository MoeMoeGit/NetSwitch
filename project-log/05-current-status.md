## 已完成

- 托盘常驻，托盘右键菜单可直接选择方案。
- 主窗口可新建、编辑、重命名、删除和激活方案。
- 内置 `DHCP（默认）` 方案。
- 首次启动可读取当前静态 IP 并导入为「自定义」方案。
- 通过 JSON 保存方案、设置和窗口位置。
- 通过 `route print` + PowerShell 自动识别当前优先网卡。
- 通过 `netsh` 写入 DHCP / 静态 IP / DNS。
- 托盘图标运行时根据状态用 Qt 动态绘制。
- 构建期通过 Pillow 生成 ico/png 资源。
- PyInstaller 单文件 exe 打包和 Inno Setup 安装包脚本。
- GitHub Actions tag 发布流程。
- 启动时自检管理员权限，不足则 UAC 重启。
- 滚动日志诊断，路径 `%AppData%\NetSwitch\netswitch.log`。
- netsh 失败信息标准化并传回 UI。
- 写入后等待改为最多 5 秒轮询（每 500ms 检测）。
- 网络写入增加全局互斥锁，托盘和主窗口入口避免并发切换。
- 二次启动可恢复隐藏到托盘的主窗口。
- 应用成功但无网关时进入 warning 状态，不再显示为正常。
- 回滚命令逐条检查，回滚失败会传回 UI 错误消息。
- 开机自启 Run 路径加引号，写入失败时提示用户并回滚勾选状态。
- 备用 DNS 增加格式校验，方案模式切换时清理不适用字段。
- 配置文件损坏时先备份为 `profiles.invalid.<timestamp>.json` 再重建默认配置。
- README 构建安装包命令已改为 `iscc scripts/installer_output.iss`。
